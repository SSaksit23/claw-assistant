## üéØ Key Components

### The Six Specialized Agents

1. **Assignment Agent** - Central coordinator that receives user requests and delegates tasks
2. **Accounting Agent** - Manages financial records and expense order creation
3. **Data Analysis Agent** - Retrieves and analyzes booking and sales data
4. **Market Analysis Agent** - Conducts competitive and market research
5. **Admin Agent** - Handles administrative tasks and record management
6. **Executive Agent** - Aggregates data and generates executive reports

### Technology Stack

- **AI Framework**: CrewAI
- **Language Model**: OpenAI (GPT-4 / GPT-4 Turbo)
- **Web Automation**: ClawBot
- **User Interface**: LINE Messaging API
- **Data Processing**: Python (Pandas, NumPy)
- **Visualization**: Matplotlib / Plotly
